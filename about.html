Weather Whisper 2.0

A smart and interactive weather platform that helps you explore the world’s weather in a fun and visual way.
It combines real-time weather updates, maps, and a rotating 3D globe — turning data into an experience.


Features

  • 3D Rotating Globe → Explore the Earth in motion.
  • Real-time Weather Updates → Get accurate temperature, wind speed, and conditions.
  • Location Detection → Automatically fetch weather for your current location.
  • Interactive Map → View your city’s weather on a detailed map.
  • Animated Starry Background → Enjoy a calming space-like ambiance.
  • Smooth Transitions → Seamless switch from globe view to weather map.
  • Simple Navigation → Quick access to the About page and weather input.

Technology Used

  • HTML & CSS – for structure and styling of the website.
  • JavaScript – for dynamic interactivity and animations.
  • Three.js – to create and render the 3D interactive globe.
  • Leaflet.js – for displaying detailed, zoomable world maps.
  • OpenWeather API – to fetch live weather data.
